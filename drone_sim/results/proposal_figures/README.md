# 基金申请图像说明文档

## 图像清单

### 核心展示图（推荐使用）

#### 1. proposal_obstacle_avoidance.png
**用途**: 系统能力展示、避障效果演示
**内容**:
- 左侧：3D视角展示无人机在复杂建筑环境中的三条避障轨迹
- 右侧：俯视图对比A*最优路径与RRT*实际飞行路径
- 标注了障碍物（4排建筑）、20m宽可穿越缝隙、起点终点
- 三条轨迹：黑色A*最优(71m)、蓝色RRT* Y=20(78m)、红色RRT* Y=-25(91m)

**亮点**:
- 直观展示系统在真实复杂环境中的避障能力
- 对比最优算法，展示实际性能差距
- 彩色编码清晰，适合PPT和海报

#### 2. proposal_pipeline.png
**用途**: 技术架构说明、系统设计展示
**内容**:
- 6个核心模块的技术管线：感知→增量建图→ESDF→能量感知RRT*→滚动规划→飞行控制
- 红色虚线箭头标注反馈回路（Receding Horizon）
- 底部标注关键性能指标：150ms周期、100%成功率

**亮点**:
- 完整展示从感知到控制的闭环系统
- 突出"能量感知"和"滚动规划"两大创新点
- 性能数据增强说服力

---

#### 3. proposal_benchmark_summary.png
**用途**: 算法性能验证、实验数据展示
**内容**:
- 左图：三场景下的路径最优性对比（Path Ratio）
- 中图：消融实验 - 4种代价函数权重配置的能耗对比
- 右图：A* vs RRT*路径长度对比（含10次运行的误差棒）

**亮点**:
- 定量数据支撑，展示算法有效性
- 消融实验证明能量感知的必要性
- 误差棒展示实验严谨性（10次重复实验）

## 关键数据摘要（可用于文字说明）

### 系统性能
- **实时性**: 150ms感知-规划-控制周期
- **成功率**: 100%（多场景测试）
- **地图更新**: 40ms增量建图
- **路径规划**: 50ms局部RRT*规划

### 算法性能
- **开阔场景**: RRT*路径仅比A*最优多0.9%（Scenario B）
- **复杂场景**: RRT*路径比A*最优多15-31%（Scenario A/C）
- **能量优化**: Balanced配置在所有场景均为最优或接近最优
- **稳定性**: 10次重复实验，标准差<2m（开阔场景）

### 实际飞行验证
- **飞行距离**: 70m导航任务
- **飞行时间**: 78.6秒
- **迭代次数**: 12次滚动规划
- **最终误差**: 0.36m
- **能耗**: 7588.7J (2.1Wh, 电池消耗2.8%)
