# 无人机自主避障系统 - 实验结果汇报材料

## 项目概述

基于 AirSim 仿真平台的无人机自主飞行与避障系统，采用 **OctoMap + ESDF + RRT*** 技术路线实现 3D 路径规划。

## 三、各图片内容说明

### receding_horizon_result.png（最新核心成果）
- **内容**：滚动规划飞行的最终结果
- **展示**：
  - 红色点云：检测到的障碍物
  - 绿色线：已执行的飞行轨迹
  - 蓝色虚线：当前规划路径
  - 橙色点：无人机当前位置
  - 紫色星：目标位置
- **技术亮点**：增量式建图 + 滚动规划实现大型障碍物绕行

### paper_visualization_final.png（论文级可视化）
- **布局**：2行3列，5个子图
- **子图说明**：
  1. 3D Map + Path：三维占据地图和规划路径
  2. ESDF XY Slice：水平切片的距离场（红=近障碍物，绿=远离障碍物）
  3. ESDF XZ Slice：垂直切片的距离场
  4. RRT* Tree：搜索树的生长过程
  5. Top View：俯视图叠加ESDF热力图
- **适用场景**：论文插图、答辩PPT

### obstacle_avoidance_test.png
- **内容**：避障功能验证测试
- **展示**：无人机检测到前方障碍物并成功绕行

### 深度图toESDF.png
- **内容**：感知到建图的流程验证
- **展示**：深度图如何转换为ESDF距离场

## 四、技术参数

| 参数 | 值 |
|------|-----|
| 体素大小 | 0.5m |
| 栅格尺寸 | 120×120×40 (60m×60m×20m) |
| RRT*迭代次数 | 3000 |
| 安全边距 | 1.5m |
| 局部规划范围 | 8m |
| 路径执行比例 | 50% |
| 飞行速度 | 2.0 m/s |
| 规划周期 | ~100-150ms |
