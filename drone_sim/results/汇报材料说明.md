# 无人机自主避障系统 - 实验结果汇报材料

## 项目概述

基于 AirSim 仿真平台的无人机自主飞行与避障系统，采用 **OctoMap + ESDF + RRT*** 技术路线实现 3D 路径规划。

## 各阶段说明

### 阶段1：感知测试 (`1_感知测试/`)

验证深度相机感知和建图流程。

| 文件 | 说明 |
|------|------|
| `depth_visualization.png` | 原始深度图的2D热力图显示 |
| `depth_3d_view.png` | 深度图转换为3D点云的可视化 |
| `深度图测试1.png` | 早期深度图测试 |
| `深度图toESDF.png` | **重要**：展示深度图→体素栅格→ESDF的完整流程 |

**汇报要点**：用 `深度图toESDF.png` 解释感知→建图的技术原理

---

### 阶段2：单次规划 Phase 1 (`2_单次规划_Phase1/`)

单帧深度图构建地图 + 一次性路径规划。

| 文件 | 说明 |
|------|------|
| `path_planning_result.png` | 单次RRT*规划结果，展示搜索树和最终路径 |
| `obstacle_avoidance_test.png` | 避障功能验证测试 |

**局限性**：单次感知存在遮挡问题，无法绑过大型障碍物

---

### 阶段3：滚动规划 Phase 2 (`3_滚动规划_Phase2/`) ★核心成果

增量式建图 + 滚动规划，解决遮挡问题。

| 文件 | 说明 |
|------|------|
| `receding_horizon_result.png` | **最重要**：滚动规划最终结果，展示完整避障轨迹 |
| `receding_horizon_video.mp4` | 滚动规划动态过程，展示地图逐步构建 |
| `airsim_flight_final.png` | AirSim仿真环境中的真实飞行结果 |
| `airsim_flight_visualization.mp4` | AirSim飞行过程录像 |

**技术亮点**：
- 增量式建图：多帧累积，解决遮挡
- 滚动规划：局部目标 + 部分执行 + 重规划
- 成功绑过大型障碍物

---

### 阶段4：论文可视化 (`4_论文可视化/`)

用于论文插图和答辩PPT的高质量可视化。

| 文件 | 说明 |
|------|------|
| `paper_visualization_final.png` | **论文级**：5子图完整展示技术流程 |
| `paper_visualization.mp4` | 论文级动画，展示规划过程 |
| `simple_3d_final.png` | 简洁版3D可视化 |
| `simple_3d_visualization.mp4` | 简洁版动画 |

**paper_visualization_final.png 子图说明**：
1. 左上 - 3D占据地图 + 规划路径
2. 中上 - ESDF XY切片（俯视距离场）
3. 右上 - ESDF XZ切片（侧视距离场）
4. 左下 - RRT*搜索树生长过程
5. 右下 - 俯视图 + ESDF热力图叠加

## 技术参数速查

| 参数 | Phase 1 | Phase 2 |
|------|---------|---------|
| 建图方式 | 单帧 | 增量累积 |
| 规划方式 | 一次性 | 滚动规划 |
| 体素大小 | 0.5m | 0.5m |
| 栅格尺寸 | 80×80×40 | 120×120×40 |
| 局部范围 | - | 8m |
| 执行比例 | 100% | 50% |
| 安全边距 | 0.8m | 1.5m |
| 规划周期 | ~100ms | ~100-150ms |
